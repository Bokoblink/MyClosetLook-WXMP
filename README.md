# MyClosetLook - 线上衣橱小程序

这是一款专为个人衣物管理和穿搭搭配设计的微信小程序。它能帮助你将实体衣橱数字化，轻松记录每一件衣物，并为的穿搭提供灵感与回顾。

## ✨ 主要功能一览

-   **我的衣橱 (衣物管理)**
    -   添加衣物：上传照片，填写名称、分类、季节、备注等详细信息。
    -   标签：根据衣物类别（上衣、下裙、配饰）自动显示并支持选择对应的属性标签（如袖型、领型、裙型等）。
    -   详细尺寸记录：为每件衣物添加精确的尺寸数据。
    -   随时编辑与移除：方便地更新衣物信息或将其从衣橱中删除。

-   **穿搭灵感 (穿搭组合)**
    -   自由搭配：从你的数字衣橱中挑选衣物（上衣、下裙、配饰），创造属于你自己的独特穿搭。
    -   穿搭封面图：你可以为每套穿搭上传一张专属的“模特图”，如果未上传，系统将从搭配的衣物中选取图片作为封面。

-   **每日穿搭 (穿搭展示)**
    -   瀑布流浏览：在首页以美观的瀑布流形式展示你所有的穿搭记录，一目了然。
    -   季节筛选：根据不同季节快速筛选穿搭，便于你在换季时回顾和参考。

-   **标签管理**
    -   提供一个便捷的页面，让你统一管理和维护衣物的分类属性标签。

## ⚙️ 技术核心

-   **前端**：微信小程序原生开发 (WXML, WXSS, JavaScript)。
-   **数据存储**：目前数据主要存储在本地。

## 🚀 开始使用

1.  **获取项目代码**：克隆或下载本项目的代码到你的本地。

2.  **微信开发者工具**：
    -   使用微信开发者工具打开项目，请确保选择项目根目录。
    -   请在 `project.config.json` 文件中，将 `appid` 修改为你自己的小程序 AppID。

3.  **配置本地信息**：
    -   进入 `miniprogram/` 目录，复制 `config.example.js` 并重命名为 `config.js`。打开新的 `config.js` 文件，填入你的**管理员OpenID**（此项主要用于某些特定功能，如本地管理标签）。

4.  **运行项目**：
    -   点击开发者工具的“编译”按钮，即可在模拟器或真机上预览和运行你的线上衣橱小程序。
